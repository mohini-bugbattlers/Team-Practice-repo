# Prathmesh Roadlines Driver App

Mobile app for drivers to manage trips and upload documents.

## Features
- Trip execution
- Document uploads (Empty Tanker Receipt, Delivery Receipt, etc.)
- GPS tracking
- Status updates
- Trip history
- Integration with admin and manager panels

## Tech Stack
- React Native / Next.js
- TypeScript
- Node.js/Express backend
- GPS integration
- Camera integration for document uploads
