# Prathmesh Roadlines Manager Panel

Manager panel for handling driver actions when drivers can't use the app.

## Features
- Perform driver actions
- Upload documents on behalf of drivers
- Update trip statuses
- Communicate with drivers
- Report issues
- Integration with admin and driver panels

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Node.js/Express backend
- Database integration
