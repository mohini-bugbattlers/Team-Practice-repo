# Prathmesh Roadlines Admin Panel

Admin panel for managing transport requests, users, and operations.

## Features
- Dashboard with overview
- Manage companies, vehicle owners, drivers, managers
- Assign trips and monitor progress
- Document verification
- Payment processing
- Reports generation
- Integration with other panels

## Tech Stack
- Next.js 14
- TypeScript
- Tailwind CSS
- Node.js/Express backend
- Database integration
